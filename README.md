# World Bank Data Pipeline Project

## Overview
This project demonstrates a simple and reproducible data pipeline for extracting, cleaning, and analyzing economic indicators from the World Bank API.

## Objectives
- Retrieve real-world economic data using APIs
- Clean and structure raw datasets
- Build a simple data pipeline
- Prepare data for analysis

## Data Source
- World Bank API (GDP per capita)

## Methodology
- API data extraction
- Data cleaning and preprocessing
- Structuring datasets for analysis

## Skills Demonstrated
- Python data processing
- API integration
- Data pipeline design
- Reproducible workflows

## How to Run
1. Install dependencies:
   pip install pandas requests
2. Run:
   python pipeline.py

## Why This Project
This project reflects real-world data challenges and demonstrates how structured pipelines can support data-driven decision-making in development and policy contexts.
